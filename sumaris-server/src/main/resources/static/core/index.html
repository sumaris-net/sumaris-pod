Tools

  • GraphiQL: data access query editor, that use the GraphQL API (HTTP Rest);
  • GraphQL subscription: a subscription query editor, that use the GraphQL subscription API (WebSocket);
  • --
  • SparQL editor: build and run your SparQL query, using YasGUI;
  • Ontology graph viewer: visualize an ontology, using WebOVOWL;
  • Ontology export: export to ontologies database content (schema, entities), using several output format (RDF, JSON-LD, ...);
  • - Web Semantic (PoC) -
  • Taxon Search: search taxon using ontologies;
  • Organization Search: search organization using ontologies;
